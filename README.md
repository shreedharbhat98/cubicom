## To run this application

* `Clone this repository`
* `npm install`

Install dependencies :

* `axios`
* `redux`
* `react-router-dom`
* `react-redux`
* `redux-thunk`


Run command

`npm start`

The app will open a new tab running on http://localhost:3000
